Search Books:

Starting with this page add the following functionality:

Searching a list of book titles by typing keywords in the input field.

Result items for the search should update instantly as characters are being typed.

Hints:
  • The full list of book titles to search through is stored in a variable named books.
  • You can make use of the onkeyup event handler.
  • You can use string1.indexOf(string2) to check for presence of string2 within string1 (returns -1 if not present)
